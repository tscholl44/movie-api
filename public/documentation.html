Movie API Endpoint Documentation

Business Logic URL HTTP Method Request body data format Response body data format
Get All Movies /movies GET None JSON object of movie array
Get Movies by Movie Title /movies/:title GET None JSON object of movie
Get Movies by Genre /movies/genre/:genreName GET None JSON object with genre name
Get Movies by Director's Name /movies/directors/:directorName GET None JSON object about director's movies
Create a New User /users POST JSON object with name and favoriteMovies. Example:
"{ "name": "mary", "favoriteMovies": [] }"		 Confirmation of new user added
Update a User Name /users/:id PUT JSON object with user info. Example:
"{ "id": 2, "name": "mary", "favoriteMovies": [] }"		 Confirmation of update
Add a Favorite Movie /users/:id/:movieId POST None Updated list of favorite movies
Remove a Movie From Favorites /users/:id/:movieId DELETE None Confirmation of movie removal
Delete a User /users/:id DELETE None Confirmation of user removal
