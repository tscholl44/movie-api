Create a New User /users POST JSON object with name and favoriteMovies. Example:
"{ "_id": "67366d678ca0b29fbf0d819a", "userId": "1", "name": "Alice Johnson", "birthday": "1990-05-15", "favoriteMovies": [] }" 		Confirmation of new user added
Update a User Name /users/:name PUT JSON object with user info. Example:
"{ "_id": "67366d678ca0b29fbf0d819a", "userId": "1", "name": "Alice Johnson", "birthday": "1990-05-15T00:00:00.000Z", "favoriteMovies": [] }" 		Confirmation of update
Add a Favorite Movie /users/:name/favoriteMovies/:_id POST None Updated user JSON object list of favorite movies Example:
"{ "_id": "67366d678ca0b29fbf0d819a", "userId": "1", "name": "Alice Johnson", "birthday": "1990-05-15", "favoriteMovies": [ "673660c78ca0b29fbf0d8190", "673662448ca0b29fbf0d8191", "673666bd8ca0b29fbf0d8195", "6736649c8ca0b29fbf0d8193" ] }"
Remove a Movie From Favorites /users/:name/favoriteMovies/:_id DELETE None Confirmation of movie removal
Delete a User /users/:name DELETE None Confirmation of user removal
